Orbital

Satellite data monitoring dashboard for aerospace analytics

Overview

Orbital is a data monitoring platform designed for tracking satellite telemetry, mission status, and environmental data in real time. The goal was to design a clean and scalable dashboard system for aerospace operators.

Problem

Aerospace teams work with complex telemetry systems and fragmented monitoring tools. The existing workflow required switching between multiple systems, slowing down mission analysis and increasing cognitive load.

Solution

I designed a unified dashboard that centralizes satellite tracking, system health monitoring, and mission analytics into one structured interface.

Design System

  • Modular card-based layout
  • Minimal typography hierarchy
  • High-contrast data visualization
  • Scalable grid system
